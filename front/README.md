# 高端企业官网模板

基于 Vue 3 + Vite + Naive UI 的高端企业官网模板，适用于制造业、科技企业、品牌服务商、集团公司等。

## 核心功能

### 首页
- 品牌 Banner
- 核心业务板块
- 企业优势
- 客户案例轮播
- 新闻动态
- 联系方式

### 关于我们
- 企业简介
- 发展历程
- 组织架构
- 核心团队
- 荣誉资质
- 企业文化

### 业务模块
- 业务分类导航
- 业务详情页（图文/视频）
- 解决方案
- 服务流程

### 案例模块
- 案例列表（按行业/类型筛选）
- 案例详情（背景/方案/成果）
- 案例下载

### 互动模块
- 在线留言表单
- 客服弹窗
- 预约咨询
- 招商加盟页面

### 基础功能
- 多语言切换（中/英）
- SEO 优化（标题/关键词/描述）
- 暗黑模式
- 适配大屏
- 响应式设计

## 技术栈

- **框架**: Vue 3
- **构建工具**: Vite
- **UI 组件库**: Naive UI
- **路由**: Vue Router
- **状态管理**: Pinia
- **国际化**: Vue I18n
- **样式**: SCSS

## 快速开始


## 开发建议

### 数据对接
当前使用静态数据，如需对接 CMS 或后端 API，请：
1. 修改 `src/views/` 下的各个页面组件
2. 替换静态数据为 API 调用
3. 添加错误处理和加载状态

### 样式定制
- 修改 `src/assets/styles/variables.scss` 中的主题颜色和样式变量
- 根据需要调整 `common.scss` 中的通用样式

### 国际化扩展
- 在 `src/i18n/` 目录下添加新的语言包
- 在组件中使用 `$t()` 函数进行国际化
